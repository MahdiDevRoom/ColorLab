ColorLab.js
Documents

ColorLab.js is a lightweight script that provides color tools for designers, such as color mixing and color harmonies.

Installation

To install, simply add the class code to your project, or download the script and include it in your HTML document. 


<script src="path/to/colorlab.js"></scripts>

now call it:


const C = new ColorLab();

Convert

ColorLab.js allows you to convert colors between multiple formats including Keyword, HEX, HSL, HSV, RGB, and CMYK with ease. 


C.toHex('RoyalBlue'); // #4169E1
C.toRgb('RoyalBlue'); // rgb(65 105 225)
C.toHsl('RoyalBlue'); // hsl(225 73% 57%)
C.toHsv('RoyalBlue'); // hsv(225 71% 88%)
C.toCmyk('RoyalBlue'); // cmyk(71% 53% 0% 12%)
