ColorLab
A Lightweight Javascript Utility For Colors

ColorLab.js is a lightweight script that provides color tools for designers, such as color mixing and color harmonies.

Installation

To install, simply add the class code to your project, or download the script and include it in your HTML document. 


<script src="path/to/colorlab.js"></scripts>

In JavaScript: 


const c = new ColorLab();

c.toHex('Spring Green'); // '#00FF7F'
c.toHsl('Spring Green'); // 'hsl(150 100% 50%)'
c.toRgb('Spring Green'); // 'rgb(0 255 127)'
c.toHsv('Spring Green'); // 'hsv(150 100% 100%)'
c.toCmyk('Spring Green'); // 'cmyk(100% 0% 50% 0%)'

c.isColor('b LU e vioLET'); // true
c.normalize('b LU e vioLET'); // 'BlueViolet'

c.mixColor('black','white'); // '#7F7F7F'
