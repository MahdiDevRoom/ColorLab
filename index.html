ColorLab.js
A Powerful Color Utility Library

A modern, lightweight JavaScript library for advanced color manipulation and conversion in web projects.

Key Features:

  • Convert between all color formats: HEX, RGB, HSL, HSV, CMYK
  • Automatic color format detection
  • Generate color shades, tints, and tones
  • Create color harmonies (analogous, complementary, triadic, etc.)
  • Calculate color contrast ratios for accessibility
  • Find nearest web-safe colors
  • Color mixing and blending capabilities
  • Full TypeScript support
